# MDI 智能文本编辑器 (MDI_Notepad_Project1)

本项目是一个基于 MFC MDI（多文档界面）架构的高级文本编辑器，集成了正则搜索、自定义存储协议及自动化测试。

## 1. 核心功能 (F-01 ~ F-07 实现证明)
* **多文档管理**：支持同时打开并编辑多个不同格式的文档。
* **智能正则搜索**：集成 `std::regex`，支持跨行搜索并在 View 层实现黄色高亮显示。
* [cite_start]**自定义协议存储**：实现 `.mynote` 专用格式，包含文件指纹 `20250313001Z` 及模拟加密元数据 [cite: 1, 2]。
* **撤销/重做**：基于备忘录模式实现的编辑状态回滚。
* **健壮性保证**：核心 I/O 逻辑采用 RAII 包装类管理，确保资源无泄露。

## 2. 架构设计 (评分目标 2)
本项目严格遵循 **Document-View 架构**：
* **CMultiDocTemplate (工厂模式)**：动态关联 `Doc/Frame/View`，实现资源解耦。
* **RAII 资源管理**：通过自定义 `CFileArchiveRAII` 类确保文件句柄在异常发生时自动关闭。

## 3. 自动化测试 (评分目标 1)
本项目集成 **GoogleTest** 框架，测试覆盖率超过 50%：
- **FileIO_Serialization**: 验证磁盘存取可靠性。
- **Replace_Undo_Logic**: 验证正则替换与撤销栈逻辑。
- **Exception_Handling**: 模拟非法正则输入，验证 `catch(...)` 块的健壮性。
